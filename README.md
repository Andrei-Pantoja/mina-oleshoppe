🛍️ Mina OleShoppe

A simple e-commerce website built to showcase and manage products in an online shop environment.

🌐 Live Demo

Check it out here:
👉 https://andrei-pantoja.github.io/mina-oleshoppe/

📌 Features
🛒 Product listing with categories
🔍 Easy browsing and filtering
🖼️ Product images display
⚡ Clean and responsive UI
🛠️ Admin/product management (if connected to backend)

🧑‍💻 Tech Stack
HTML
CSS
JavaScript
(Optional: Firebase for backend & storage)
🚀 Purpose

This project was created as a practice e-commerce site to demonstrate:

Frontend development skills
UI/UX design basics
Product management functionality
